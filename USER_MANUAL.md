# ส่วนที่ 1: คู่มือสำหรับผู้ใช้งานทั่วไป (User)

## 1.1 การสมัครสมาชิกและเข้าสู่ระบบ

### การสมัครสมาชิกใหม่

1. **เข้าสู่หน้าสมัครสมาชิก**
   - คลิกปุ่ม "สมัครสมาชิก" ที่มุมบนขวาของหน้าเว็บ
   - หรือไปที่ URL: `/register`

2. **กรอกข้อมูลการสมัคร**
   - **อีเมล**: กรอกอีเมลที่ใช้งานจริง (จะใช้สำหรับเข้าสู่ระบบ)
   - **รหัสผ่าน**: ตั้งรหัสผ่านอย่างน้อย 6 ตัวอักษร
   - **ยืนยันรหัสผ่าน**: กรอกรหัสผ่านซ้ำอีกครั้ง

3. **ยืนยันการสมัคร**
   - คลิกปุ่ม "สมัครสมาชิก"
   - ระบบจะส่งอีเมลยืนยันไปยังอีเมลที่ลงทะเบียน
   - คลิกลิงก์ในอีเมลเพื่อยืนยันบัญชี

### การเข้าสู่ระบบ

1. **ไปที่หน้า Login**
   - คลิกปุ่ม "เข้าสู่ระบบ" ที่มุมบนขวา
   - หรือไปที่ URL: `/login`

2. **กรอกข้อมูล**
   - **อีเมล**: อีเมลที่ใช้สมัครสมาชิก
   - **รหัสผ่าน**: รหัสผ่านของบัญชี

3. **คลิก "เข้าสู่ระบบ"**
   - หากสำเร็จ จะถูกนำไปยังหน้า Dashboard

### ลืมรหัสผ่าน

1. **คลิก "ลืมรหัสผ่าน?"** ที่หน้า Login
2. **กรอกอีเมล** ที่ใช้สมัครสมาชิก
3. **ตรวจสอบอีเมล** - ระบบจะส่งลิงก์รีเซ็ตรหัสผ่าน
4. **คลิกลิงก์** และตั้งรหัสผ่านใหม่

---

## 1.2 การเปรียบเทียบประกัน

### ขั้นตอนการเปรียบเทียบ

1. **เลือกประเภทประกัน**

   จากหน้าแรก เลือกประเภทประกันที่ต้องการ:
   - **ประกันสุขภาพ** - คุ้มครองค่ารักษาพยาบาล
   - **ประกันชีวิต** - คุ้มครองกรณีเสียชีวิต
   - **ประกันรถยนต์** - คุ้มครองรถยนต์

2. **กรอกข้อมูลสำหรับค้นหา**

   **สำหรับประกันสุขภาพ/ชีวิต:**
   - **อายุ**: กรอกอายุผู้เอาประกัน (ปี)
   - **งบประมาณ**: เลือกงบประมาณเบี้ยประกันต่อปี
     - ไม่เกิน 5,000 บาท
     - 5,000 - 10,000 บาท
     - 10,000 - 20,000 บาท
     - 20,000 - 50,000 บาท
     - มากกว่า 50,000 บาท

   **สำหรับประกันรถยนต์:**
   - **ประเภทรถ**: รถเก๋ง, กระบะ, SUV ฯลฯ
   - **ยี่ห้อ/รุ่น**: เลือกยี่ห้อและรุ่นรถ
   - **ปีจดทะเบียน**: ปีที่รถจดทะเบียน

3. **ดูผลการค้นหา**

   คลิก "ค้นหาแผนประกัน" เพื่อดูรายการแผนประกันที่ตรงตามเงื่อนไข

### หน้าผลการค้นหา

**การจัดเรียงผลลัพธ์:**
- ราคาต่ำ-สูง (ค่าเริ่มต้น)
- ราคาสูง-ต่ำ
- ทุนประกันสูงสุด
- คะแนนสูงสุด

**ข้อมูลที่แสดงในแต่ละแผน:**
- ชื่อแผนประกัน
- บริษัทประกัน
- เบี้ยประกันต่อปี
- ทุนประกัน
- ความคุ้มครองหลัก
- คะแนนบริษัท

### การเลือกแผนเพื่อเปรียบเทียบ

1. **คลิกที่การ์ดแผนประกัน** ที่ต้องการเปรียบเทียบ
2. **เลือกได้สูงสุด 3 แผน** - จะมีแถบแสดงจำนวนที่เลือก
3. **คลิก "เปรียบเทียบ"** เพื่อดูตารางเปรียบเทียบแบบละเอียด

### ตารางเปรียบเทียบ

แสดงข้อมูลเปรียบเทียบแบบคู่ขนาน:

| รายการ | แผน A | แผน B | แผน C |
|--------|-------|-------|-------|
| บริษัท | - | - | - |
| เบี้ยประกัน/ปี | - | - | - |
| ทุนประกัน | - | - | - |
| ระยะรอคอย | - | - | - |
| อายุที่รับ | - | - | - |
| คะแนน | - | - | - |
| ความคุ้มครอง | - | - | - |

## 1.3 การบันทึกแผนประกัน

### วิธีบันทึกแผน

1. **ต้องเข้าสู่ระบบก่อน** - หากยังไม่ล็อกอิน ระบบจะแจ้งเตือน

2. **คลิกไอคอนหัวใจ** ที่มุมของการ์ดแผนประกัน
   - หัวใจสีเทา = ยังไม่บันทึก
   - หัวใจสีแดง = บันทึกแล้ว

3. **ดูแผนที่บันทึก**
   - ไปที่ Dashboard > แผนที่บันทึก
   - หรือ URL: `/dashboard/saved`

### การจัดการแผนที่บันทึก

**ที่หน้า "แผนที่บันทึก":**
- ดูรายการแผนทั้งหมดที่บันทึกไว้
- คลิก "ดูรายละเอียด" เพื่อดูข้อมูลแผนเต็ม
- คลิก "ลบ" เพื่อนำออกจากรายการบันทึก
- สามารถเลือกหลายแผนเพื่อเปรียบเทียบได้

## 1.4 การแชร์ผลการเปรียบเทียบ

### วิธีสร้างลิงก์แชร์

1. **เลือกแผนประกัน** ที่ต้องการเปรียบเทียบ (2-3 แผน)
2. **คลิก "เปรียบเทียบ"** เพื่อเปิดตารางเปรียบเทียบ
3. **คลิกปุ่ม "แชร์"** (ไอคอนลิงก์)
4. **ลิงก์จะถูกคัดลอก** ไปยัง Clipboard อัตโนมัติ

### การใช้ลิงก์แชร์

- ส่งลิงก์ให้คนอื่นผ่าน LINE, Email, SMS
- ผู้รับลิงก์จะเห็นตารางเปรียบเทียบเดียวกัน
- ไม่จำเป็นต้องล็อกอินเพื่อดูลิงก์แชร์
- ลิงก์ไม่มีวันหมดอายุ

### รูปแบบลิงก์

```
https://easy-insurance-pi.vercel.app/share/xxxxxxxx
```

## 1.5 การ Export PDF

### วิธี Export เป็น PDF

1. **เลือกแผนประกัน** ที่ต้องการเปรียบเทียบ
2. **คลิก "เปรียบเทียบ"** เพื่อเปิดตารางเปรียบเทียบ
3. **คลิกปุ่ม "Export PDF"** (ไอคอนดาวน์โหลด)
4. **รอสักครู่** ระบบจะสร้างไฟล์ PDF
5. **ไฟล์จะดาวน์โหลดอัตโนมัติ**

### เนื้อหาใน PDF

- หัวข้อ "Easy Insurance - ผลการเปรียบเทียบ"
- ประเภทประกันที่เปรียบเทียบ
- ตารางเปรียบเทียบแบบละเอียด
- วันที่และเวลาที่สร้าง
- ข้อความแจ้งเตือนให้ตรวจสอบกับบริษัทประกัน

### ข้อแนะนำ

- PDF รองรับภาษาไทยเต็มรูปแบบ
- เหมาะสำหรับส่งให้ลูกค้าประกอบการตัดสินใจ
- สามารถพิมพ์ออกมาเป็นเอกสารได้

## 1.6 Dashboard ส่วนตัว

### การเข้าถึง Dashboard

- คลิกที่ชื่อผู้ใช้มุมบนขวา > "Dashboard"
- หรือไปที่ URL: `/dashboard`

### เมนูใน Dashboard

#### 1. ภาพรวม (Dashboard)
แสดงสถิติส่วนตัว:
- จำนวนแผนที่บันทึก
- จำนวนการเปรียบเทียบ
- แผนประกันล่าสุดที่ดู

#### 2. แผนที่บันทึก (Saved Plans)
- รายการแผนประกันที่บันทึกไว้
- จัดเรียงตามวันที่บันทึก
- ลบหรือเปรียบเทียบได้

#### 3. ประวัติการเปรียบเทียบ (History)
- บันทึกการเปรียบเทียบทั้งหมด
- แสดงวันที่ ประเภทประกัน และแผนที่เปรียบเทียบ
- สามารถดูลิงก์แชร์ที่สร้างไว้

#### 4. ตั้งค่า (Settings)
- แก้ไขข้อมูลส่วนตัว (ชื่อ, เบอร์โทร)
- เปลี่ยนรหัสผ่าน
- ออกจากระบบ

# ส่วนที่ 2: คู่มือสำหรับผู้ดูแลระบบ (Admin)

> **หมายเหตุ**: ผู้ใช้คนแรกที่สมัครสมาชิกจะได้รับสิทธิ์ Admin อัตโนมัติ
> Admin สามารถเปลี่ยน Role ของผู้ใช้อื่นได้ผ่านหน้า "จัดการผู้ใช้"

## 2.1 ภาพรวม Admin Dashboard

### การเข้าถึง Admin

- คลิกเมนู "Admin" ที่แถบด้านบน
- หรือไปที่ URL: `/admin`
- **ต้องมีสิทธิ์ Admin** จึงจะเข้าถึงได้

### หน้า Dashboard หลัก

แสดงสถิติภาพรวม:

| สถิติ | คำอธิบาย |
|------|---------|
| แผนประกันทั้งหมด | จำนวนแผนประกันในระบบ |
| Leads ใหม่ | ลูกค้าที่สนใจรอติดต่อกลับ |
| ผู้ใช้งาน | จำนวนสมาชิกทั้งหมด |
| Commission เดือนนี้ | ยอดค่าคอมมิชชั่นเดือนปัจจุบัน |

### เมนู Admin

1. **Dashboard** - ภาพรวมระบบ
2. **แผนประกัน** - จัดการแผนประกัน
3. **Leads** - จัดการลูกค้าที่สนใจ
4. **ผู้ใช้งาน** - จัดการสมาชิก
5. **Commission** - ติดตามค่าคอมมิชชั่น
6. **แจ้งเตือน** - รายการติดตามลูกค้า
7. **Analytics** - วิเคราะห์ข้อมูล
8. **ตั้งค่า** - ตั้งค่าระบบ

## 2.2 การจัดการแผนประกัน

### เข้าถึงหน้าจัดการ

- Admin > แผนประกัน
- URL: `/admin/plans`

### รายการแผนประกัน

แสดงตารางแผนประกันทั้งหมด:
- ชื่อแผน
- บริษัท
- ประเภท (สุขภาพ/ชีวิต/รถยนต์)
- เบี้ยประกัน
- สถานะ (Active/Inactive)
- ปุ่มแก้ไข/ลบ

### การเพิ่มแผนประกันใหม่

1. **คลิกปุ่ม "+ เพิ่มแผน"**

2. **กรอกข้อมูลแผน:**

   **ข้อมูลพื้นฐาน:**
   - ชื่อแผน (เช่น "สุขภาพ Plus 500,000")
   - บริษัทประกัน (เลือกจากรายการ)
   - ประเภทประกัน (สุขภาพ/ชีวิต/รถยนต์)

   **ข้อมูลราคา:**
   - เบี้ยประกันต่อปี (บาท)
   - ทุนประกัน (บาท)

   **เงื่อนไข:**
   - อายุขั้นต่ำ (ปี)
   - อายุสูงสุด (ปี)
   - ระยะรอคอย (วัน)

   **ความคุ้มครอง:**
   - เพิ่มรายการความคุ้มครองทีละข้อ
   - เช่น "ค่ารักษาพยาบาลผู้ป่วยใน"
   - เช่น "ค่าห้องต่อวัน 5,000 บาท"

3. **คลิก "บันทึก"**

### การแก้ไขแผนประกัน

1. คลิกปุ่ม "แก้ไข" ที่แถวของแผนที่ต้องการ
2. แก้ไขข้อมูลตามต้องการ
3. คลิก "บันทึก"

### การปิด/เปิดใช้งานแผน

- แผน Active = แสดงในหน้าค้นหา
- แผน Inactive = ไม่แสดงในหน้าค้นหา
- คลิก Toggle ที่คอลัมน์ "สถานะ" เพื่อเปลี่ยน

### การลบแผนประกัน

1. คลิกปุ่ม "ลบ" ที่แถวของแผน
2. ยืนยันการลบ
3. **คำเตือน**: ข้อมูลที่ลบไม่สามารถกู้คืนได้

## 2.3 การจัดการ Leads

### Leads คืออะไร?

Leads คือรายชื่อลูกค้าที่สนใจแผนประกัน โดยกรอกข้อมูลติดต่อไว้ในระบบ

### เข้าถึงหน้า Leads

- Admin > Leads
- URL: `/admin/leads`

### สถานะ Lead

| สถานะ | ความหมาย | สี |
|-------|---------|-----|
| new | Lead ใหม่ รอติดต่อ | น้ำเงิน |
| contacted | ติดต่อแล้ว | เหลือง |
| quoted | ส่งใบเสนอราคาแล้ว | ส้ม |
| converted | ปิดการขายสำเร็จ | เขียว |
| closed | ปิด Lead (ไม่สนใจ) | เทา |

### การดูรายละเอียด Lead

คลิกที่แถวของ Lead เพื่อเปิด Modal รายละเอียด:

**ข้อมูลที่แสดง:**
- ชื่อ-นามสกุล
- เบอร์โทรศัพท์
- แผนประกันที่สนใจ
- ข้อความเพิ่มเติม
- วันที่ส่ง Lead
- สถานะปัจจุบัน

### การอัพเดทสถานะ

1. เปิด Modal รายละเอียด Lead
2. เลือกสถานะใหม่จาก Dropdown
3. สถานะจะอัพเดทอัตโนมัติ

### การบันทึกโน้ต (Customer Notes)

**วิธีเพิ่มโน้ต:**
1. เปิด Modal รายละเอียด Lead
2. ไปที่ส่วน "บันทึกการติดต่อ"
3. พิมพ์โน้ตในช่อง (เช่น "โทรไปแล้ว ลูกค้าขอใบเสนอราคา")
4. คลิก "เพิ่มโน้ต"

**ประโยชน์:**
- บันทึกประวัติการติดต่อกับลูกค้า
- ตัวแทนอื่นสามารถอ่านได้
- ช่วยในการติดตาม Lead

### การตั้งวันนัดติดตาม (Follow-up)

1. เปิด Modal รายละเอียด Lead
2. ไปที่ช่อง "วันนัดติดตาม"
3. เลือกวันที่ต้องการนัดติดตาม
4. Lead จะปรากฏในหน้า "แจ้งเตือน"

### การโทรหาลูกค้า

- คลิกปุ่ม "โทร" ที่แถวของ Lead
- ระบบจะเปิดแอพโทรศัพท์ (บนมือถือ)
- หรือแสดงเบอร์สำหรับโทร (บน Desktop)

## 2.4 การจัดการผู้ใช้งาน

### เข้าถึงหน้าจัดการผู้ใช้

- Admin > ผู้ใช้งาน
- URL: `/admin/users`

### รายการผู้ใช้

แสดงตารางผู้ใช้ทั้งหมด:
- อีเมล
- ชื่อ (ถ้ามี)
- Role (บทบาท)
- วันที่สมัคร
- สถานะ

### Role (บทบาท)

| Role | คำอธิบาย | สิทธิ์การเข้าถึง |
|------|---------|-----------------|
| admin | ผู้ดูแลระบบ | เข้าถึงได้ทุกหน้า รวมถึง /admin |
| agent | ตัวแทนขาย | ดู Leads และ Analytics ได้ |
| customer | ลูกค้าทั่วไป | เปรียบเทียบประกัน, Dashboard ส่วนตัว |

### การเปลี่ยน Role ผู้ใช้

1. คลิก Dropdown ที่คอลัมน์ "Role"
2. เลือก Role ใหม่
3. ยืนยันการเปลี่ยนแปลง

**ข้อควรระวัง:**
- ควรมี Admin อย่างน้อย 1 คน
- การเปลี่ยนจาก Admin เป็น Customer จะทำให้ผู้ใช้นั้นเข้าถึง Admin ไม่ได้

### การค้นหาผู้ใช้

- ใช้ช่องค้นหาที่ด้านบนตาราง
- ค้นหาได้จาก อีเมล หรือ ชื่อ

## 2.5 Commission Tracker

### เข้าถึงหน้า Commission

- Admin > Commission
- URL: `/admin/commission`

### ภาพรวม Commission

แสดงสถิติ:

| รายการ | คำอธิบาย |
|--------|---------|
| Commission ทั้งหมด | ยอดรวมค่าคอมมิชชั่นทั้งหมด |
| Commission เดือนนี้ | ยอดค่าคอมมิชชั่นเดือนปัจจุบัน |
| Leads ที่ปิดการขาย | จำนวน Lead ที่ status = converted |
| อัตรา Commission เฉลี่ย | ค่าเฉลี่ยเปอร์เซ็นต์คอมมิชชั่น |

### การคำนวณ Commission

```
Commission = เบี้ยประกัน × อัตรา Commission (%)
```

**ตัวอย่าง:**
- เบี้ยประกัน: 12,000 บาท
- อัตรา Commission: 15%
- Commission = 12,000 × 0.15 = 1,800 บาท

### รายการ Commission

ตารางแสดง:
- ชื่อลูกค้า
- แผนประกันที่ซื้อ
- เบี้ยประกัน
- อัตรา Commission
- จำนวน Commission
- วันที่ปิดการขาย

### การตั้งค่าอัตรา Commission

1. ไปที่ Admin > ตั้งค่า
2. ส่วน "อัตรา Commission"
3. ตั้งค่าอัตราตามประเภทประกัน:
   - ประกันสุขภาพ: X%
   - ประกันชีวิต: Y%
   - ประกันรถยนต์: Z%

## 2.6 Follow-up Reminders

### เข้าถึงหน้าแจ้งเตือน

- Admin > แจ้งเตือน
- URL: `/admin/reminders`

### ประเภทการแจ้งเตือน

| ประเภท | คำอธิบาย | สี |
|--------|---------|-----|
| เลยกำหนด | วันนัดผ่านไปแล้ว ยังไม่ได้ติดต่อ | แดง |
| วันนี้ | นัดติดตามวันนี้ | เหลือง |
| กำลังจะถึง | นัดติดตามในอนาคต | เขียว |

### หน้า Reminders

**ส่วนบนสุด - สถิติ:**
- จำนวน Lead ที่เลยกำหนด
- จำนวน Lead ที่ต้องติดตามวันนี้
- จำนวน Lead ที่กำลังจะถึงกำหนด

**ปุ่มกรอง:**
- "ทั้งหมด" - แสดงทุกรายการ
- "เลยกำหนด" - เฉพาะที่เลยกำหนด
- "วันนี้" - เฉพาะวันนี้
- "กำลังจะถึง" - เฉพาะที่ยังไม่ถึงกำหนด

### การจัดการ Reminder

**แต่ละ Reminder แสดง:**
- ชื่อลูกค้า
- เบอร์โทรศัพท์
- แผนประกันที่สนใจ
- วันที่นัดติดตาม
- โน้ตล่าสุด

**ปุ่มการดำเนินการ:**
- **โทรหา** - เปิดแอพโทรศัพท์
- **ดูรายละเอียด** - ไปหน้า Lead Detail

### การติดตามลูกค้า

1. ตรวจสอบรายการ "เลยกำหนด" ทุกวัน
2. โทรหาลูกค้าตามรายการ
3. บันทึกผลการโทรในโน้ต
4. ตั้งวันนัดติดตามครั้งถัดไป (ถ้าจำเป็น)
5. เปลี่ยนสถานะ Lead ตามความเหมาะสม

## 2.7 Analytics

### เข้าถึงหน้า Analytics

- Admin > Analytics
- URL: `/admin/analytics`

### กราฟและสถิติ

#### 1. Lead Overview
- จำนวน Lead ทั้งหมด
- Lead ใหม่เดือนนี้
- อัตราการปิดการขาย (Conversion Rate)

#### 2. Lead ตามสถานะ
แผนภูมิวงกลมแสดงสัดส่วน:
- New / Contacted / Quoted / Converted / Closed

#### 3. Lead ตามประเภทประกัน
แผนภูมิแท่งแสดง:
- จำนวน Lead แยกตามประกันสุขภาพ/ชีวิต/รถยนต์

#### 4. แนวโน้ม Lead รายเดือน
กราฟเส้นแสดง:
- จำนวน Lead ใหม่ในแต่ละเดือน
- เทียบกับเดือนก่อนหน้า

#### 5. Top Plans
แผนประกันที่ได้รับความสนใจสูงสุด

## คำถามที่พบบ่อย (FAQ)

### Q: ลืมรหัสผ่านทำอย่างไร?
A: ไปที่หน้า Login > คลิก "ลืมรหัสผ่าน?" > กรอกอีเมล > ตรวจสอบอีเมลและคลิกลิงก์รีเซ็ต

### Q: ทำไม PDF ไม่แสดงภาษาไทย?
A: ระบบใช้ฟอนต์ Noto Sans Thai รองรับภาษาไทย หากมีปัญหา ลองรีเฟรชหน้าแล้ว Export ใหม่

### Q: แชร์ลิงก์หมดอายุไหม?
A: ไม่หมดอายุ ลิงก์สามารถใช้งานได้ตลอด

### Q: ทำไมเข้า Admin ไม่ได้?
A: ต้องมีสิทธิ์ Admin เท่านั้น ติดต่อผู้ดูแลระบบเพื่อเปลี่ยน Role

### Q: Commission คำนวณอย่างไร?
A: Commission = เบี้ยประกัน × อัตรา Commission ที่ตั้งไว้ในระบบ
